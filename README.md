### 기획 배경
#### 🚩 문제 의식
"단순 정보 전달에 그치는 획일화된 템플릿"
결혼식 문화가 변화하며 종이 청첩장보다 모바일 청첩장의 중요성이 커지고 있습니다. 하지만 현재 시장의 모바일 청첩장은 단순히 '사진만 갈아 끼우는' 획일화된 템플릿 수준에 머물러 있습니다. 이는 정보 전달의 기능은 수행하지만, 신랑신부만의 아이덴티티를 담아내기에는 한계가 존재했습니다.

#### 🗣️ 사용자 인터뷰
"우리만의 서사가 담긴 디자인을 원해요"
실제 모바일 청첩장을 제작해본 경험자와 인터뷰를 진행하여 다음과 같은 구체적인 문제점을 발견했습니다.

- 아이덴티티의 부재: "기존 업체들의 디자인은 너무 뻔해요."  
  영화관에서 처음 만난 우리만의 추억을 담아 만들고 싶었지만, 시중 템플릿에는 그런 기능이 없어 결국 디자인 툴을 배워 직접 제작해야 했습니다.  
- 커스텀의 진입장벽: "특별하게 만들고 싶지만, 너무 어려워요."  
  남들과 다른 청첩장을 만들고 싶었지만, 포토샵 같은 전문 툴을 다룰 줄 몰라 결국 포기하고 흔한 템플릿을 선택했습니다.

#### 💡 해결책: "AI로 누구나 쉽게 완성하는 커스텀 디자인"
"AI 청첩장(가제)"는 생성형 AI 기술을 도입하여 디자인 툴 장벽을 없애고, 누구나 '자신만의 이야기'가 담긴 청첩장을 만들 수 있는 솔루션을 제안합니다.

- **텍스트/컨셉 기반 AI 생성**  
"따뜻한", "현대적인" 등 원하는 키워드만 입력하면 AI가 최적의 레이아웃과 디자인을 자동으로 제안합니다.

- **직관적인 UX**  
복잡한 툴 학습 없이, 단계별(Step-by-Step) 입력 방식을 통해 정보 입력부터 디자인 생성까지 간편하게 완료할 수 있습니다.

✨ 엔터테인먼트 콘텐츠화: 단순 초대를 넘어 하객에게 즐거움을 줄 수 있도록 3D 캐릭터 변환 및 영화 포스터 패러디 기능을 제공합니다.
